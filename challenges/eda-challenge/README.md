# EDA Challenge - The Green River Affair

## Folder layout

- `EpochFail_EDA_challenge.ipynb` - main report notebook (Colab-ready)
- `data/` - datasets used by the notebook
- `docs/` - mission briefing and assignment instructions

## Run in Google Colab

1. Open `EpochFail_EDA_challenge.ipynb` in Colab.
2. Run all cells from top to bottom.

If Colab cannot find the datasets, make sure the repository is available in the runtime (for example by cloning it) and that `data/` sits next to the notebook.
