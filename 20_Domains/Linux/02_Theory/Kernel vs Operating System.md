# Kernel vs Operating System
type: theory
domain: Linux
tags: #kernel #os
---
Het onderscheid tussen de kernel en het complete besturingssysteem is essentieel voor het begrip van hoe computers werken.

## Kernel
De kern die hardware beheert, processen plant en geheugen beheert.

## Operating System
Het complete pakket inclusief kernel, bibliotheken, utilities en interface (shell).
