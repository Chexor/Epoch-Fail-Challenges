# Neural Network Models
#concept #ml

Neural Network Models groepeert de concepten rond neurale netwerken en deep learning als rijkere modelfamilie binnen machine learning.

## Korte kern

- vormt de kapstok voor neural networks en deep learning
- gaat over gelaagde modellen voor complexe patronen
- verbindt klassieke ML met rijkere modelarchitecturen

## 1. Wat is het kernprobleem?
Hoe modelleer je complexe, niet-lineaire patronen die moeilijk te vatten zijn met eenvoudige modellen?

## 2. Intuitieve uitleg
Wanneer lineaire of eenvoudige modellen tekortschieten, gebruik je neurale netwerken met meerdere lagen om rijkere representaties te leren.

## 5. Toepassing
Bij beeldherkenning of andere complexe patroonproblemen zijn neural network models vaak krachtiger dan klassieke lineaire modellen.

## 6. Begripsafbakening en verbanden
- **Kernonderscheid:** deze note groepeert modelarchitecturen, niet de frameworks waarmee je ze implementeert.
- **Veelvoorkomende misvatting:** deep learning verwarren met een tool in plaats van een modelfamilie.
