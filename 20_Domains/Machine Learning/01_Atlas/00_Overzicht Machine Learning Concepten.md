# Overzicht Machine Learning Concepten
#concept #ml

Overzicht Machine Learning Concepten als master-atlas voor het vak "Machine Learning Fundamentals".

> Deze note is het startpunt voor alle ML-gerelateerde concepten.

## Korte kern

- overkoepelend overzicht van ML
- vertakt naar specifieke atlas-notes per subdomein
- helpt de samenhang tussen fundamenten en modellen te begrijpen

## Groot plaatje
De ML-concepten in dit domein vallen in drie grote blokken uiteen:

1. **Fundamenten**: Basisbegrippen, paradigma's en probleemtypes ([[Atlas - Machine Learning Fundamenten]])
2. **Modellen & Algoritmen**: Concrete implementaties voor voorspelling en extractie ([[Atlas - Modellen en Algoritmen]])
3. **Training & Evaluatie**: Optimalisatie, verliesfuncties en modelvalidatie ([[Atlas - Training en Evaluatie]])

Deze blokken samen vormen de volledige keten van data naar bruikbaar model.
