# Feature
#concept #ml

Een `feature` is een specifieke invoervariabele of eigenschap die door het model wordt gebruikt om een voorspelling te maken.

## Korte kern

- modelinput
- variabele die informatie bevat voor voorspelling

## Waarom is dit belangrijk?
De kwaliteit van de gekozen features bepaalt grotendeels de prestatie van het model.

## Toepassing
In een tabel is een feature een kolom die wordt gebruikt als input voor de voorspelling van de target.
