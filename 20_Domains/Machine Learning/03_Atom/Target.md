# Target
#concept #ml

De `target` (of `label`) is de grootheid die het model moet voorspellen op basis van de input-features.

## Korte kern

- te voorspellen output
- de 'ground truth' in supervised learning

## Waarom is dit belangrijk?
Het is het doel van het leerproces; het model optimaliseert zijn parameters om de fout op deze waarde te minimaliseren.

## Toepassing
In een tabel is de target de kolom die je wil voorspellen.
