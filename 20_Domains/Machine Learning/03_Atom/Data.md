# Data
#concept #ml

`Data` (of de `dataset`) is de verzameling van alle beschikbare voorbeelden (`examples`) die gebruikt worden voor training, validatie of test van een model.

## Korte kern

- volledige verzameling van data
- verzameling van `examples`

## Waarom is dit belangrijk?
Zonder data kan een model niet leren. De kwaliteit en representativiteit van de data bepalen hoe goed een model generaliseert.

## Formele structuur
- Dataset `D = {(x_i, y_i)}`
