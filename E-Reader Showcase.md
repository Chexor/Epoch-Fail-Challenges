### :LiTerminal: Code Blocks

Code blocks maintain the E-ink aesthetic with a muted background and gray text, reducing the harsh contrast typical of dark-mode code editors.

```python
def calculate_focus_score(distractions: int) -> float:
    # A simple function to demonstrate code block styling
    base_focus = 100.0
    impact = distractions * 5.5
    
    return max(0.0, base_focus - impact)
```
