# Learning Paradigms
#concept #ml

Learning Paradigms groepeert de grote families van machine learning op basis van het soort feedback dat een model krijgt tijdens het leren.

## Korte kern

- groepeert supervised, unsupervised en reinforcement learning
- helpt bepalen welk soort leerprobleem je voor je hebt
- vormt de kapstok voor de basisfamilies van ML

## 1. Wat is het kernprobleem?
Hoe deel je machine learning op in fundamentele families zodat duidelijk wordt welk soort data en feedback een model krijgt?

## 2. Intuitieve uitleg
Niet elk model leert op dezelfde manier. Soms krijgt het juiste antwoorden te zien, soms alleen ruwe data, en soms leert het via beloningen en straffen. Deze verschillende leerlogica's noem je learning paradigms.

## 3. Formele structuur
De drie centrale paradigma's in deze cursus zijn:

- [[Supervised Learning]]
- [[Unsupervised Learning]]
- [[Reinforcement Learning]]

Deze paradigma's bepalen onder meer:

- welk type trainingsdata beschikbaar is
- welke doelen het model heeft
- welke algoritmes logisch zijn

## 4. Snelle vergelijking

- [[Supervised Learning]] = leren met labels
- [[Unsupervised Learning]] = leren zonder labels
- [[Reinforcement Learning]] = leren via beloning en interactie

## 5. Toepassing
Wanneer je eerst wil bepalen of een ML-probleem gelabelde data heeft, of eerder draait om clustering of interactie, dan denk je in termen van learning paradigms.

## 6. Begripsafbakening en verbanden
- **Kernonderscheid:** dit is een overkoepelende notie voor leerfamilies, niet zelf een concreet model.
- **Veelvoorkomende misvatting:** supervised, unsupervised en reinforcement learning studeren als losse termen zonder hun gemeenschappelijke rol binnen machine learning te zien.
