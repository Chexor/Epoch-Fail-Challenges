# Machine Learning vs Deep Learning
#concept #ml

## Wat wordt hier vergeleken?
- [[Machine Learning]]
- [[Deep Learning]]

## Kernverschil
`Machine Learning` is het brede domein van modellen die leren uit data. `Deep Learning` is een specifiek subdomein binnen machine learning dat neurale netwerken met meerdere lagen gebruikt.

## Vergelijking

| Aspect | Machine Learning | Deep Learning |
| --- | --- | --- |
| Reikwijdte | breed overkoepelend domein | subdomein binnen ML |
| Modeltypes | lineaire modellen, KNN, K-Means, neurale netwerken | vooral diepe neurale netwerken |
| Structuur | kan simpel of complex zijn | meerdere hidden layers |
| Feature engineering | vaak belangrijk | vaak automatisch geleerd |
| Datahonger | vaak matig | vaak groot |
| Rekenkracht | vaak beperkter | vaak zwaarder |
| Interpretatie | vaak makkelijker | vaak moeilijker, black box |
| Typische fout | denken dat ML altijd deep learning is | denken dat deep learning volledig los staat van ML |

## Wanneer gebruik je wat?
- **Machine Learning:** als je een breder kader nodig hebt of met klassieke modellen werkt zoals [[Linear Regression]], [[K-Nearest Neighbors (KNN)]] of [[K-Means]].
- **Deep Learning:** als de patronen heel complex zijn en neurale netwerken met meerdere lagen een duidelijke meerwaarde bieden, bijvoorbeeld bij beelden of spraak.

## Waarom is dit onderscheid belangrijk?
Veel studenten gebruiken beide termen alsof ze synoniemen zijn. Dat is fout. `Deep Learning` is een krachtige, maar specifiekere familie binnen het grotere veld van `Machine Learning`.
