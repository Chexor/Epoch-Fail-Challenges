# Kansverdelingen

**Wat is het?**
Een functie die beschrijft hoe de kansen verdeeld zijn over de mogelijke waarden van een toevalsvariabele. Het is een theoretisch model van de werkelijkheid.

**Belangrijke Voorbeelden:**

1.  **Binomiale Verdeling:**
    -   **Gebruik:** Voor situaties met een vast aantal herhalingen en twee mogelijke uitkomsten (succes/mislukking).
    -   **Voorbeeld:** Hoe groot is de kans op 3 keer "kop" bij 5 worpen met een munt?

2.  **Normale Verdeling (Gauss-curve):**
    -   **Gebruik:** Beschrijft heel veel natuurlijke fenomenen (lengte, gewicht, IQ).
    -   **Kenmerken:** Symmetrisch, klokvormig. Het gemiddelde, de mediaan en de modus vallen samen.

**Waarom is dit belangrijk?**
Als we weten welke verdeling onze data volgt, kunnen we krachtige statistische uitspraken en voorspellingen doen.
