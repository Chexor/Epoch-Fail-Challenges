# Soorten Variabelen

**Wat is het?**
Een variabele is een kenmerk dat je meet. Er zijn twee hoofdsoorten:

1.  **Kwalitatieve Variabelen:** Beschrijven een eigenschap of categorie.
    -   **Voorbeeld:** Oogkleur (blauw, bruin), diploma (bachelor, master).

2.  **Kwantitatieve Variabelen:** Drukken een hoeveelheid uit in een getal.
    -   **Discreet:** Telbaar, enkel gehele getallen (bv. aantal studenten).
    -   **Continu:** Kan elke waarde binnen een bereik aannemen (bv. lengte, gewicht).

**Waarom is dit belangrijk?**
Het type variabele bepaalt welke grafieken en berekeningen zinvol zijn.
