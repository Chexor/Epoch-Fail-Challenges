# Meetniveaus

**Wat is het?**
Meetniveaus geven aan hoeveel informatie een variabele bevat. Van weinig naar veel info:

1.  **Nominaal:** Categorieën zonder volgorde (bv. provincies).
2.  **Ordinaal:** Categorieën mét een logische volgorde (bv. opleidingsniveau: lager, middelbaar, hoger).
3.  **Interval:** Getallen met een vaste afstand, maar geen echt nulpunt (bv. temperatuur in Celsius).
4.  **Ratio:** Getallen met een vaste afstand én een betekenisvol nulpunt (bv. leeftijd, gewicht).

**Waarom is dit belangrijk?**
Het meetniveau bepaalt welke wiskundige bewerkingen je mag uitvoeren. Je kan het gemiddelde van "Antwerpen" en "Limburg" niet berekenen.
