# Toevalsvariabelen

**Wat is het?**
Een variabele waarvan de waarde een numerieke uitkomst is van een toevalsproces. We weten de waarde niet op voorhand.

**Twee Soorten:**
1.  **Discrete Toevalsvariabele:** Kan een eindig, telbaar aantal waarden aannemen.
    -   *Voorbeeld:* Het aantal ogen bij een worp met een dobbelsteen (1, 2, 3, 4, 5, of 6).
    -   Beschreven door een **kansfunctie**.

2.  **Continue Toevalsvariabele:** Kan elke waarde binnen een bepaald interval aannemen.
    -   *Voorbeeld:* De exacte lengte van een willekeurige student.
    -   Beschreven door een **dichtheidsfunctie**.

**Belangrijke Eigenschappen:**
-   **Verwachtingswaarde (E[X]):** Het gemiddelde op lange termijn.
-   **Variantie (Var(X)):** Hoeveel de waarden gemiddeld afwijken van de verwachtingswaarde.
