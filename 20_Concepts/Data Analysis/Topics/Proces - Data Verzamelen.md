# Data Verzamelen

**Wat is het?**
De fase waarin je de data verzamelt die nodig is om je onderzoeksvraag te beantwoorden.

**Kernconcepten:**
-   **[[Populatie en Steekproef|Populatie vs. Steekproef]]:** Bepaal over welke groep je een uitspraak wilt doen (de populatie) en welke subgroep je daadwerkelijk zult meten (de steekproef).
-   **Sampling Methoden:** Hoe selecteer je de steekproef? Willekeurig (aselect) is het ideaal om [[Representativiteit]] te garanderen.
-   **Data Bronnen:** Enquêtes, databases, sensoren, webscraping, etc.

**Waarom is dit belangrijk?**
De kwaliteit van je data bepaalt de kwaliteit van je analyse. "Garbage in, garbage out." Een foute of vertekende (biased) dataverzameling leidt onvermijdelijk tot foute conclusies, hoe geavanceerd je analyses ook zijn.
