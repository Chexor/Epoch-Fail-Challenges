# Overzicht Linear Algebra
#concept #linal

## Doel van dit overzicht

Deze note groepeert de kernconcepten van lineaire algebra zodat je sneller ziet hoe matrices, bewerkingen, inverteerbaarheid en oplossingsmethoden samenhangen.

## Groot plaatje

Lineaire algebra draait niet alleen om rekenen met tabellen van getallen. Het gaat om een samenhangend systeem waarin je leert:
- hoe informatie in matrixvorm wordt voorgesteld
- welke bewerkingen geldig zijn
- wanneer een matrix omkeerbaar is
- hoe je stelsels en transformaties structureel analyseert

Deze map-note is dus een leerkaart van de belangrijkste verbanden binnen dit domein.
