This item explains what the dot product is and how the linear regression model applies it to make predictions.

- There is an Excel-file "house_data.xlsx" with the dataset.
- Excel-file "house_data - demo.xlsx" shows how predictions are made in Excel by first multiplying each feature (column) by its weight and then taking the sum of all features.
- Notebook "Dot_product.ipynb" explains and shows how it is done in NumPy and Pandas using function or method dot().
