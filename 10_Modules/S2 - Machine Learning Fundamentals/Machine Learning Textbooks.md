- _Grokking Machine Learning_ by Luis Serrano (2021)
- _Hands‑On Machine Learning_ (3ed) by Aurélien Géron (2023)

The topic on deep learning is based on the first chapters of the following textbook:

- _Deep Learning with Python_ (2ed) by François Chollet (2021)
