# 01 - Introductie tot Linux en Unix

Een overzicht van de geschiedenis, werking en impact van Unix en Linux, en de open-source filosofie erachter.

## Wat is Unix?
Unix is een familie van besturingssystemen, voor het eerst ontwikkeld in 1969 (oorspronkelijk als Multics). Het vormt de fundering voor veel moderne systemen, waaronder macOS (gebaseerd op FreeBSD).

**Belangrijkste kenmerken:**
- Multi-user en multi-tasking
- Hiërarchisch bestandssysteem
- Eenvoudige, krachtige Command-Line Interface (CLI)
- De filosofie: *"Everything is a file"*

## Besturingssysteem (OS) vs. Kernel
- **Besturingssysteem (OS):** Het complete softwarepakket, inclusief de kernel, system utilities, libraries en de user interface (zoals een shell of desktop environment).
- **Kernel:** De absolute kern van het OS. Verantwoordelijk voor:
  - Hardware resource management
  - Memory management & process scheduling
  - File systems
  - Het aanbieden van system calls

## Het GNU Project & Vrije Software
Opgericht in 1983 door Richard Stallman. GNU is een recursief acroniem voor *"GNU's Not Unix"*. Het doel was om een volledig vrij Unix-achtig besturingssysteem te bouwen. 
Onderdelen die hieruit voortkwamen: GCC, GNU Coreutils, GNU Bash en GDB.

**De Vier Vrijheden van Free Software:**
0. Vrijheid om het programma te draaien voor elk doel.
1. Vrijheid om te bestuderen hoe het programma werkt en het aan te passen.
2. Vrijheid om kopieën te verspreiden.
3. Vrijheid om gewijzigde versies te verspreiden.

**GPL (GNU General Public License):** Dit is een *Copyleft* licentie (de tegenhanger van Copyright, wat restrictiever is). Het zorgt ervoor dat de software én al haar afgeleiden altijd vrij blijven. Wie code aanpast en deelt, moet dit onder dezelfde GPL-voorwaarden met inbegrip van de broncode doen.

## De Geboorte van Linux
Linus Torvalds, een Finse informaticastudent, begon in augustus 1991 aan een eigen kernel als hobbyproject. Hij haalde inspiratie uit **MINIX** (een educatief OS van Andrew Tanenbaum met een microkernel-design). 
- Oorspronkelijke naam voor het project was "Freax", maar werd later "Linux" ("Linus' Unix").
- In combinatie met de GNU-tools (zoals bash en gcc) ontstond een krachtig, volwaardig en gratis besturingssysteem.

## Linux Distributies (Distro's)
Een distributie is een compleet OS dat rondom de Linux-kernel is gebouwd, elk met een eigen package manager, configuratie en filosofie.

**Drie bekende families:**
1. **Debian Family:** Gekend om stabiliteit en free software principes (bv. Debian, Ubuntu).
2. **Red Hat Family:** Sterk gericht op de enterprise markt (bv. RHEL, Fedora, CentOS).
3. **SUSE Family:** Enterprise en community oplossingen (bv. openSUSE, SUSE Linux Enterprise).

## Linux in de Praktijk
Linux is dominant aanwezig in de professionele IT-infrastructuur:
- **Servers & Cloud:** Het fundament voor webservers, databases, cloud platforms en container orchestration (zoals Docker en Kubernetes).
- **Supercomputers:** Meer dan 95% van de supercomputers ter wereld draait op Linux.
- **Top Techbedrijven:** Partijen als Google, Amazon (AWS), Netflix en SpaceX draaien hun kritieke systemen nagenoeg volledig op Linux.

## Waarom Linux leren?
Linux-vaardigheden zijn extreem gewild en essentieel voor rollen in DevOps, Cloud Computing en Cybersecurity. Het biedt je bovendien volledige controle, krachtige CLI-tools en een stabiel fundament voor software development.
