# Introduction to Linux - Labomgeving (Linux Server School)

## Doel

Voor veel labo's werk je op de schoolserver via SSH.

## Inloggen op de server

Gebruik in je terminal (PowerShell, CMD of Linux terminal) een van deze commando's:

```bash
ssh rNR@93.115.18.104
```

of

```bash
ssh rNR@linux.vives.live
```

## Betekenis van `rNR`

- `rNR` staat voor je persoonlijk r-nummer of studentenlogin.

## Snelle checklist

- [ ] Terminal openen op je host.
- [ ] SSH-commando uitvoeren met je eigen login.
- [ ] Eerste login bevestigen (host key) indien gevraagd.
- [ ] Verifiëren dat je op de juiste machine zit met `whoami` en `hostname`.
