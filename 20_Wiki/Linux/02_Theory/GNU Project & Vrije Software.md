# GNU Project & Vrije Software
type: theory
domain: Linux
tags: #gnu #freesoftware
---
Het GNU Project (1983) streeft naar een volledig vrije Unix-achtige omgeving.

## Kern
- **Componenten**: GCC, Bash, Coreutils, Make, GDB.
- **De Vier Vrijheden**:
    1. Vrijheid om het programma te gebruiken voor elk doel.
    2. Vrijheid om de werking te bestuderen en aan te passen.
    3. Vrijheid om kopieën te herdistribueren.
    4. Vrijheid om verbeterde versies te verspreiden.
