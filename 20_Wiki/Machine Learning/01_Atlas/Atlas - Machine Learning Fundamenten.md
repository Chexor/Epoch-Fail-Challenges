# Atlas - Machine Learning Fundamenten
#concept #ml

Atlas-note voor de fundamenten van Machine Learning.

> Deze note ordent de basisbegrippen, paradigma's en probleemtypes in Machine Learning.

## Korte kern

- ordent ML-basisbegrippen
- verduidelijkt leerparadigma's (Supervised, Unsupervised, RL)
- dient als navigatiepunt voor ML-theorie en -toepassingen

## Waarom bestaat deze conceptgroep?
Machine Learning verschuift het paradigma: in plaats van expliciete regels te definiëren, leert het systeem patronen uit data om intelligent gedrag te vertonen.

## Groot plaatje
ML is een fundamentele tak van AI waarbij data centraal staat om modellen te bouwen die generaliseren naar nieuwe situaties.
