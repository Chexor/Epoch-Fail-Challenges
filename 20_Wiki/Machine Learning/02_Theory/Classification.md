# Classification
#concept #ml

**Classification** is een type [[Supervised Learning|supervised learning]] waarbij het doel is om een input-datapunt te labelen met een vooraf gedefinieerde, discrete categorie of klasse.

## Korte kern
Classification voorspelt tot **welke categorie** iets behoort. Het antwoord is een vaste, kwalitatieve label, zoals "spam" of "geen spam", "kat" of "hond".

## 1. Wat is het kernprobleem?
Hoe kunnen we een model trainen om nieuwe, onbekende data correct in een van de vooraf bepaalde groepen of klasses in te delen?

## 2. Intuitieve uitleg
Stel je een postsorteercentrum voor. Een machine leert om brieven te herkennen als 'standaard', 'aangetekend' of 'reclame' door naar duizenden voorbeelden met de juiste labels te kijken. Classification is het proces waarbij deze machine een nieuwe brief ziet en er onmiddellijk het juiste label op plakt.

## 3. Formele structuur
- **Input:** Een datapunt met een set kenmerken (features).
- **Output:** Een discreet label uit een eindige set van mogelijke klasses.
- **Doel:** Een functie leren die de input-kenmerken mapt naar het juiste output-label.
- **Paradigma:** [[Supervised Learning]], omdat het trainingsdata met gekende labels vereist.

## 4. Snelle vergelijking
- **[[Classification]]**: Voorspelt een categorie (bv. `hond` of `kat`).
- **[[Regression]]**: Voorspelt een continue waarde (bv. `3.45 kg`).
- **[[Clustering]]**: Vindt groepen in data zonder labels.

## 5. Toepassing
- **Spamdetectie:** Een e-mail classificeren als "spam" of "geen spam".
- **Beeldherkenning:** Een afbeelding van een dier classificeren als "hond", "kat", of "vogel".
- **Medische diagnose:** Een tumor classificeren als "goedaardig" of "kwaadaardig".

## 6. Begripsafbakening en verbanden
- **Kernonderscheid:** De output is altijd een discrete categorie, geen getal op een schaal.
- **Link met andere concepten:** Is een van de twee hoofdtaken binnen [[Supervised Learning]], naast [[Regression]]. Algoritmes zoals [[K-Nearest Neighbors (KNN)]] en [[Support Vector Machine (SVM)]] worden vaak voor classificatie gebruikt.
