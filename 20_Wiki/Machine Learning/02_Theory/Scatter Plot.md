# Scatter Plot
#concept #ml

A scatter plot is een grafiek die twee numerieke variabelen als punten in een assenstelsel toont zodat je hun relatie visueel kan inschatten.

## Korte kern

- elk punt is een observatie
- toont richting, sterkte en vorm van een relatie
- helpt outliers snel zien

## 1. Wat is het kernprobleem?
Hoe zie je snel of twee numerieke variabelen samen lijken te hangen?

## 2. Intuitieve uitleg
Je zet voor elk datapunt de waarde van variabele `x` op de horizontale as en `y` op de verticale as. Het puntenpatroon toont dan meteen of er een stijgende, dalende, zwakke of geen duidelijke relatie is.

## 3. Formele structuur
- x-as = eerste numerieke variabele
- y-as = tweede numerieke variabele
- elk punt = een observatie `(x, y)`
- een ongeveer rechte puntenwolk kan wijzen op een lineaire relatie
- losse of kromme patronen kunnen wijzen op zwakke of niet-lineaire verbanden

## 4. Snelle vergelijking
- [[Scatter Plot]] = relatie tussen twee numerieke variabelen tonen
- [[Exploratory Data Analysis (EDA)]] = bredere verkenningsfase waarin scatter plots een hulpmiddel zijn
- [[Linear Regression]] = model dat een lineair verband probeert te fitten

## 5. Toepassing
Als je `aantal kamers` tegen `huisprijs` plot en de punten liggen ongeveer rond een stijgende rechte, dan is [[Linear Regression]] een redelijke kandidaat.

## 6. Begripsafbakening en verbanden
- **Kernonderscheid:** een scatter plot toont een patroon, maar berekent zelf nog geen model.
- **Veelvoorkomende misvatting:** een duidelijke puntenwolk verwarren met causaliteit.
