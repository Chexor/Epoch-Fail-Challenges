# Regression
#concept #ml

**Regression** is een type [[Supervised Learning|supervised learning]] waarbij het doel is om een continue, numerieke waarde te voorspellen voor een gegeven input-datapunt.

## Korte kern
Regression voorspelt een **specifieke hoeveelheid** of een getal. Het antwoord is een kwantitatieve waarde op een continue schaal, zoals de prijs van een huis, de temperatuur van morgen, of de vraag naar een product.

## 1. Wat is het kernprobleem?
Hoe kunnen we een model trainen om een precieze, numerieke voorspelling te doen op basis van een set input-kenmerken?

## 2. Intuitieve uitleg
Stel je voor dat je de verkoopprijs van een huis wilt voorspellen. Je verzamelt data van andere huizen: hun grootte, aantal kamers, locatie, en de prijs waarvoor ze verkocht zijn. Regressie is het proces waarbij een model de onderliggende relatie tussen de kenmerken (grootte, kamers) en de prijs leert, zodat het voor een nieuw huis een nauwkeurige prijsschatting kan maken.

## 3. Formele structuur
- **Input:** Een datapunt met een set numerieke en/of categorische kenmerken (features).
- **Output:** Een enkele, continue numerieke waarde.
- **Doel:** Een functie leren die de input-kenmerken mapt naar de meest waarschijnlijke output-waarde.
- **Paradigma:** [[Supervised Learning]], omdat het trainingsdata met gekende numerieke 'antwoorden' vereist.

## 4. Snelle vergelijking
- **[[Regression]]**: Voorspelt een continue waarde (bv. `150.750 euro`).
- **[[Classification]]**: Voorspelt een discrete categorie (bv. `verkocht` of `niet verkocht`).

## 5. Toepassing
- **Financiën:** De toekomstige prijs van een aandeel voorspellen.
- **Weervoorspelling:** De maximale temperatuur voor morgen voorspellen.
- **Marketing:** Het aantal verwachte verkopen voor een product voorspellen.

## 6. Begripsafbakening en verbanden
- **Kernonderscheid:** De output is een getal op een continue schaal, wat betekent dat er oneindig veel mogelijke uitkomsten zijn binnen een bereik.
- **Link met andere concepten:** Is de andere hoofdtak van [[Supervised Learning]] naast [[Classification]]. Het meest bekende basisalgoritme is [[Linear Regression]].
